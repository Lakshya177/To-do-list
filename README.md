# To-do-list

📌 Overview:- 
<pre>
This is a To-Do List Web Application built using Java, Hibernate, JSP, and Servlets. 
It helps users manage tasks with CRUD operations and simple pagination.
</pre>

🚀 Features:-
<pre>
1 🧾 Task Management
2 ➕ Add Task (Title, Description)
3 📋 Show All Tasks
4 ✏️ Edit Task
5 ❌ Delete Task
</pre>
